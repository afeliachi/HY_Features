The HY_IndirectPosition feature type defines the location referenced along a logical axis, without the necessity of a geometric realization which it however may have.<br /></p><p><br /></p><p>The indirect position concept uses the flowpath as linear element  along which a position may be assigned referencing a known location, usually the catchment outlet, or a feature that can be identified as (said to be coincident with) the outlet.The indirect position is expressed as distance from that referent already located on the flowpath, e.g. the upstream or downstream end of the flowpath. <br /></p><p><br /></p><p>HY_IndirectPosition carries the <i>distanceExpression</i>, and <i>distanceDescription</i> attributes, and associates the <i>linearElement</i> and the <i>locatedReferent.</i><br /></p>
