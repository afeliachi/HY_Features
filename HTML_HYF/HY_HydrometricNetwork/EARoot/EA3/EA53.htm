topology-based reference system to simply assign a position 'along a river' using its geometric representation. The origin of the river reference system is set at the location to be placed, understood as outfall of a catchment. The geometric shape is defined by the linear flowpath representing the catchment between the origin and an identified reference location upstream or downstream of the origin. <br /></p><p><br /></p><p>The position on flowpath is provided as distance from the located start of the flowpath at origin and end at refrecne location towards the flowpath is directed. HY_RiverReferenceSystem shall support one axis.<br /></p><p><br /></p><p>Note: Each catchment has its own Reference System which can be combined as necessary using catchment topology. Each Reference System must have one outfall (origin) and one FlowPath (shape).<br /></p>
