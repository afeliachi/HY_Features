special catchment wherein all waters flow to a single common outlet. A dendritic catchment is permanently connected to others in a dendritic network. This class denotes the catchment as the link between inflow and outflow which allows to describe a dendritic network by upstream-downstream relationships. This concept requires a stable identifier that is not merely a function of an arbitrary delineation of the surface, and that catchments are delineated within a tree hierarchy.
