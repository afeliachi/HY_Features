linear reference system used to assign a position along a Flowpath (i.e. along  the linear representation of the catchment) whereby origin is located at the outfall of the catchment that corresponds to the relevant feature of interest, e.g. a water monitoring point.  The shape is defined by the flowpath representing the catchment corresponding to the reference location. The position along the flowpath is provided as distance from the origin to the reference location (i.e. the known start or end of the representing flowpath). RiverReferenceSystem shall support one axis.<br /></p><p><br /></p><p>Note: Each catchment has its own Reference System which can be combined as necessary using catchment topology. Each Reference System must have one outfall (origin) and one FlowPath (shape).<br /></p>
